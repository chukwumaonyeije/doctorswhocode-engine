Setup Checklist
Step 01 — Telegram Bot

Create Your Bot

  • Open Telegram, search @BotFather
  • Run /newbot and name it (e.g. ResearchAgentBot)
  • Copy the Bot Token
  • Add token to n8n Telegram credential
  • Send any message to your bot to activate the webhook
Step 02 — n8n Setup

Import the Workflow

  • Open n8n (Railway or local)
  • Click + New Workflow
  • Menu → Import from JSON
  • Paste research-agent-workflow.json
  • Activate the workflow toggle
Step 03 — Credentials

Wire Your Keys

  • Add Telegram credential with your Bot Token
  • Add Anthropic credential with your API key
  • Add GitHub credential (Personal Access Token, repo scope)
  • Set your GitHub username and repo name in the GitHub node
  • Create a drafts branch in your blog repo
Step 04 — Test It

First Run

  • Send any article URL to your Telegram bot
  • Watch n8n execution panel
  • Expect response in 25-45 seconds
  • Check GitHub drafts branch for .mdx file
  • Review Telegram output for all 5 content blocks
